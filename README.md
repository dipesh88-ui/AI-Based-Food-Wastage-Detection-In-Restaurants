📌 About the Project

This project is an AI-powered web application built with Next.js that helps restaurants track and reduce food wastage. It analyzes food usage patterns and provides insights to improve efficiency, reduce costs, and promote sustainability.

🚀 Features

AI-based food waste analysis

Real-time tracking system

Clean and modern UI

Data-driven insights for better decisions

Helps reduce cost and improve efficiency

⚙️ Tech Stack

Next.js

React

TypeScript

Tailwind CSS

Firebase

Genkit AI

🛠️ Installation & Setup
npm install
npm run dev
🌐 Run Locally

Open your browser and go to:

http://localhost:9002
🔑 Environment Variables

Create a .env.local file in root:

GEMINI_API_KEY=your_api_key
